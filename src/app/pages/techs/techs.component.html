Se utilizo angular 16 para el FrontEnd, aprovechando la facilidad que ofrece a la hora de modularizar los componentes y de manejar las rutas necesarias para la navegacion por la pagina.

Por otro lado, para el backend se uso nodejs v18, para crear el servidor pertinente para procesar las solicitudes entrantes por parte del cliente, y redirigirlas a la IA, ademas de esto se maneja la logica de "verdades" sobre las consultas hechas a la misma.

Para el forwarding de el puerto local hacia una red abierta en internet se uso "ngrok" util herramientas parecida a cloudflare, pero con mayores libertades en su version gratuita.

El modelo preentrenado de ia empleado es (INSERTAR) basado en Llama2, corriendo gracias al proyecto de oobabooga especificamente text-generation-webui, el proyecto en si esta creado para generar una webui con gradio, pero ademas tiene la posibilidad de levantar una api.
